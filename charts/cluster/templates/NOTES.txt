# After installation notes

Created a cluster with using following providers:

Core provider: "{{ .Values.capi.providers.core.name }}"
Infrastructure provider: "{{ .Values.capi.providers.infrastructure.name }}"
Control plane provider: "{{ .Values.capi.providers.controlPlane.name }}"
Bootstrap provider: "{{ .Values.capi.providers.bootstrap.name }}"

Command to get kubeconfig:
kubectl get secret -n {{ .Release.Namespace }} {{ include "cluster-name" . }}-kubeconfig -o jsonpath='{.data.value}' | base64 -d
